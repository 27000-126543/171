## 1. 产品概述

大型商业综合体运营管理桌面系统，面向商业综合体运营团队，实现商铺租赁、运营方案、停车场、安保排班、保洁排班、能耗监控、广告位管理及统计分析的一体化智能管理平台。
- 解决多业态综合体运营中信息分散、人工调度效率低、异常响应慢等核心痛点
- 目标用户：运营经理、招商部、安保部、保洁部、物业部，提升运营效率与决策智能化

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|

### 2.2 功能模块

1. **总览仪表盘**：核心KPI指标、今日待办、实时告警、客流趋势图、商铺状态概览
2. **商铺租赁管理**：商铺信息录入、合同管理、自动计费与账单、催缴通知
3. **运营方案管理**：自动生成每日方案、审批流程、方案推送
4. **停车场管理**：车位实时监控、自动计费、VIP预约审批
5. **安保排班**：智能排班、巡逻路线生成、异常事件派单
6. **保洁排班**：自动任务分配、超时升级通知
7. **能耗监控**：实时数据采集、阈值报警、节能建议
8. **广告位管理**：租赁排期、自动计费、到期下架
9. **统计报表**：多维度统计、Excel导出、可视化平面图与热力图

## 3. 核心流程

### 3.1 商铺租赁全流程
商户信息录入 → 合同签署 → 系统自动计算租金 → 按月生成账单 → 商户缴费确认 → 逾期自动催缴通知 → 合同到期提醒续签/退租

### 3.2 运营方案生成与审批流程
系统采集历史客流数据/节假日信息/商户活动申请 → 自动生成每日运营方案（营业时间/促销安排/动线优化）→ 运营经理审批 → 通过后推送至各商户终端

### 3.3 异常事件处置流程
异常事件检测（火警/冲突/设备故障）→ 系统自动定位最近安保人员 → 自动派单 → 安保人员现场处置 → 记录处置结果 → 事件归档

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深蓝灰(#1B2A4A) + 琥珀金(#F0A500) + 翠绿(#10B981)，体现商业综合体的专业与高端感
- **辅助色**：警报红(#EF4444)、警告橙(#F59E0B)、信息蓝(#3B82F6)、成功绿(#10B981)
- **按钮风格**：圆角6px，主按钮实心填充，次要按钮描边，危险操作红色警告
- **字体**：标题使用 Noto Sans SC Bold，正文使用 Noto Sans SC Regular，数据数字使用 DM Mono
- **布局风格**：左侧固定导航栏 + 顶部工具栏 + 主内容区，卡片式内容组织
- **图标风格**：线性图标，2px描边，与整体风格统一
- **背景**：深色模式为主(#0F172A)，搭配渐变背景与微弱噪点纹理

### 4.3 响应式设计

- 桌面优先设计，最小支持1440px宽度
- 1920px及以上充分利用空间展示更多数据面板
- 1440px以下适当收缩侧边栏，内容区保持可用

### 4.4 动效设计

- 页面切换：淡入淡出 + 轻微位移
- 数据更新：数字滚动动效
- 告警触发：红色脉冲边框 + 闪烁图标
- 卡片hover：轻微上浮 + 阴影加深
- 侧栏展开/收起：平滑过渡动画
